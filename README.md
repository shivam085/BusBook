# 🚌 Bus Booking System

A production-quality Bus Booking System with multi-city route support, interactive seat selection, Razorpay payments, PDF tickets, email notifications, wallet system, and segment-based seat inventory.

## Tech Stack
- **Frontend**: React 18, Vite, Tailwind CSS (v3), React Router v6, Axios, Context API
- **Backend**: Node.js, Express.js, Sequelize (MySQL)
- **Real-time**: Socket.IO (for seat locking)
- **Integrations**: Razorpay, Nodemailer, PDFKit
